Sesión 2: Control de Flujo

Aprende a usar el condicional if y el bucle for para controlar tu programa.

# Condicionales y Ciclos
puntos = 85

if puntos >= 70:
    print("¡Aprobado!")

for i in range(3):
    print(f"Repetición número: {i}")
